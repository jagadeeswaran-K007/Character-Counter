
# 📝 Real-Time Character Counter

A clean, minimalist, and responsive web application built with HTML, CSS, and vanilla JavaScript that tracks your character count instantly as you type. Perfect for writing social media bios, tweets, SMS messages, or any text requiring precise length constraints.

---
🔗 **Live Demo:** [https://jagadeeswaran-k007.github.io/Character-Counter/]

---

## 🚀 Key Features

* ⚡ **Real-Time Counter:** Dynamically updates the "Characters Used" and "Characters Left" counts on every keystroke.
* 🚫 **Smart Constraint:** Enforces a rigid limit of 200 characters to keep inputs concise.
* 🧼 **Quick Reset:** Features a dedicated "Clear Text" button to wipe the textarea instantly and start over.
* 📱 **Fully Responsive:** Styled to look great and work flawlessly on mobile phones, tablets, and desktop browsers.

---

## 🛠️ Tech Stack Used

* 🌐 **HTML5:** Structures the layout and form input elements.
* 🎨 **CSS3:** Delivers a modern UI, featuring clean typography, styled counters, and smooth layout elements.
* 💛 **JavaScript (ES6+):** Utilizes lag-free event listeners (`input`/`keyup`) and real-time DOM manipulation to handle logic.

---

## 🎯 Learning Outcomes

Building this micro-project helped reinforce:

* DOM manipulation and capturing real-time user inputs.
* JavaScript string properties like .length.
* CSS flexbox workflows for creating centered, clean component layouts.
